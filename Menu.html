이지모션 홈페이지를 찾아 주신 여러분께 진심으로 환영합니다.




우리는 CAE 기술과 딥러닝 기술을 활용하여 시간 영역 실시간

해석을 위한 ROM 솔루션 개발을 선도하는 기업입니다.



우리 기업의 가치는 고객과의 긴밀한 협력을 통해 고객이 기대하는 이상의 결과물을

창출하는 것에 있다고 생각합니다.

최신 시뮬레이션 기술과 해석 엔지니어링 노하우를 가진 엔지니어들이 고객과

성실하게 소통함으로써 고객과의 신뢰를 쌓고 고객의 니즈를 만족시키는 것을 최우선으로 생각합니다.



우리 기업의 열정적이고 창의적인 직원들과 CAE 기술과 딥러닝 기술을 통해 새로운

실시간 해석 분야의 가능성을 탐구하고 있으며, 고객과 함께 성장하고 발전하는 기업이

되기를 너무나 기대하고 있으며 그것을 실천하도록 노력하는 대표가 되겠습니다.

ROM Builder




ROM Builder는 ROM(Reduced Order Model)을 생성하는 SW로

컴퓨터 시뮬레이션이나 시험에서 얻는 데이터를 ROM Builder 입력하여

이지모션의 예측 기술로 ROM을 생성해 주는 툴입니다.

생성된 ROM은 실시간 해석 모델로 시스템의 물리적 특성을 실시간 예측이

필요한 분야(Virtual Sensor, HILS, Digitaltwin)에서 활용이 가능한 모델입니다.

또한, 설계에서 성능 예측 솔버로도 사용이 가능하기에 설계 자동화 프로그램에

꼭 필요한 기술로 평가 받고 있습니다.

Customization

사용자 해석 편의성을 위한 M&S 자동화 툴킷 개발




Ansys Motion 사용자 편의성을 극대화하기 위해 M&S 자동화 툴킷을 개발 지원합니다.

·수치 해석 모델 기법 개발

·친숙하고 간단한 사용자 인터페이스 개발

·모델링 자동화

·해석 결과 분석 기법 개발

·보고서 생성 자동화



자동화 툴킷을 통해 사용자는 다음과 같은 이익을 가질 수 있습니다.

·쉬운 UI/UX를 통해 어려운 해석 모델 빌드

·어려운 SW 사용 교육 불필요

·모델링 실수 최소화

·해석 시간 최소화

·해석 결과 분석 시간 최소화

Consulting

사용자의 엔지니어링 문제를 시뮬레이션을 이용해 재현하고 개선안 검증




사용자의 제품 Life Cycle 전반에 있는 공학적인 문제를 Ansys 제품을

활용하여 재현하고 문제를 해결하거나 개선할 수 있는 방안을 제안하고

시뮬레이션으로 검증하는 컨설팅을 수행하고 있습니다.



이지모션의 주요 컴퓨터 시뮬레이션 영역은 다음과 같습니다.

·다물체 동역학 해석

·NVH 해석

·다분야 연성/비연성 해석

Ansys Motion Basic




Ansys Motion을 이용한 다물체 동역학 해석 모델링 방법 및 분석 방법을 배웁니다.

다양한 예제를 통해 바디, 조인트, 하중, 접촉 요소를 이용한 다자유도 시스템의
동역학 해석 모델링 방법과 시간 영역의 해석 결과를 이용해 기계 시스템의 동특성을
분석하는 방법을 기초 교육 과정에서 배우게 됩니다.

교육 항목

1. Motion Preprocessor
2. Motion Postprocessor
3. Body Entity
4. Constraint Entity
5. Force Entity
6. Contact Entity


Ansys Motion Advanced




Ansys Motion의 기초 교육을 받으신 분들을 대상으로 Motion을 좀 더 다양하게
활용하기 위해 고급 기능을 배우게 됩니다.

바디 고유치 해석을 통한 유연체 생성 방법, 사용자 함수, 내구강도 해석, 시스템 고유치 해석,
피로해석 등 당양한 해석 방법을 고급 교육 과정에서 배우게 됩니다.

교육 항목

1. Flexible Body Entity
2. Body Eigenvalue Analysis
3. System Eigenvalue Analysis
4. EasyFlex Entity
5. Durability Analysis
6. Fatigue Analysis


Ansys Motion Drivetrain




Ansys Motion Drivetrain 툴킷을 이용하여 동력전달 시스템의 소음진동을 분석하는 방법을
드라이버트레인 교육 과정에서 배우기 됩니다.

동력전달 시스템을 구성하고 있는 기어, 베어링, 샤프트, 하우징의 모델링 방법과 RPM 및 부하 하중
입력을 통해 시스템의 화인과 래틀 진동소음 해석 및 분석 방법을 예제를 통해서 배울 수 있습니다.
여러분들의 제품이 진동소음 문제가 있다면 바로 신청하셔서 문제를 해결하시기 바랍니다.

교육 커리큘럼

1. Gear Design
2. Bearing Design
3. Shaft Design
4. Housing Design
5. FFT, STFT, Order Tracking
6. Drivetrain example
